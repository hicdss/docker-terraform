# docker-terraform

Swiss army knife for DevOps


# Tools

This image contains following software:

- terraform `0.12.24`
- aws `1.18.39`
- az `2.2.0`
- kubectl `1.17.0`
- helm `2.16.1`
- azcopy `10`

And additionally:

- vim
- curl
- jq
- git
- ssh
- make
- python
- pip
